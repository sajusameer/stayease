# 🏨 StayEase - Hotel Booking Platform

StayEase is a modern full-stack hotel booking platform where users can explore hotels, view detailed information, and book stays online. It includes authentication, role-based dashboards, hotel management, booking management, search, filtering, sorting, pagination, and an admin panel.

## 🌐 Live Demo

**Live Site:** https://stayease-4a9c.vercel.app

---

## ✨ Features

### 👤 Authentication
- Better Auth authentication
- User Registration & Login
- Role-based access (Admin/User)
- Demo Login (Admin & User)
- Protected Routes

### 🏨 Hotel Management
- Browse all hotels
- Hotel details page
- Featured hotels
- Add Hotel (Admin only)
- Manage Hotels (Admin only)
- Delete Hotel

### 🔍 Search & Filtering
- Search hotels
- Filter by Category
- Filter by Location
- Sort by:
  - Newest
  - Price Low → High
  - Price High → Low
  - Highest Rating
- Pagination

### 📅 Booking System
- Book hotel
- Dynamic total price calculation
- My Bookings page
- Booking status

### 📊 Dashboard
#### User Dashboard
- Booking statistics
- Profile
- My Bookings

#### Admin Dashboard
- Total Hotels
- Total Bookings
- Total Users
- Dynamic Charts (Recharts)

### 📱 Responsive Design
- Mobile Friendly
- Tablet Friendly
- Desktop Friendly

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/sajusameer/stayease
```

Install dependencies

```bash
npm install
```

Create a `.env.local`

```env
MONGODB_URI=YOUR_MONGODB_URI
BETTER_AUTH_SECRET=YOUR_SECRET
BETTER_AUTH_URL=http://localhost:3000
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

Run the project

```bash
npm run dev
```
